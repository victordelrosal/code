What is OpenCode & Why Use Terminal?

The difference between web UI and terminal AI coding

You've Used ChatGPT... Now Level Up

In the Hello World guide, you asked an AI to generate code, then copy-pasted it into a file. That works great for learning, but what happens when you're building real projects with dozens of files?

OpenCode is an open-source AI coding assistant that runs in your terminal. Instead of copy-pasting between browser and editor, you have a conversation with AI inside your project folder. The AI can read your files, understand your codebase, and write code directly.

Web UI vs Terminal: The Key Difference

Web UI (ChatGPT, Claude, Gemini)

  • You copy code from the AI
  • You paste into your editor
  • AI can't see your files
  • You describe your code in words
  • Great for learning concepts

Terminal (OpenCode)

  • AI reads your actual files
  • AI writes directly to your project
  • Full context of your codebase
  • No copy-paste needed
  • Built for real projects

Think of it this way

Web UI is like texting a friend who helps with code. Terminal AI is like having that friend sitting next to you, looking at your screen, and typing for you.

Why This Matters

When you say "fix the bug in my login function":

  • Web UI: You have to paste your entire file, explain where the function is, wait for a response, then manually apply changes
  • Terminal: The AI finds your login function, understands the context, and fixes it directly

This guide will get you set up with OpenCode so you can experience the difference yourself.

Detailed Step-by-Step Guide

Complete instructions for Mac and Windows

Part 1: Open Terminal (Don't Panic!)

What is Terminal?

Terminal is a text-based way to talk to your computer. Instead of clicking buttons, you type commands. It looks like something from a hacker movie, but don't worry — you'll only need a few simple commands, and we'll guide you through each one.

Think of it like texting your computer instead of tapping on icons.

Mac: Opening Terminal

  1. Hold down Cmd (the key with ⌘) and press Space
  2. A search bar appears in the middle of your screen (this is called "Spotlight")
  3. Type the word Terminal
  4. Press Enter or click on "Terminal" in the results
  5. A window opens with a blinking cursor — that's Terminal!

You'll see something like yourname@MacBook ~ % followed by a blinking cursor. This is the "command line" where you type commands.

Windows: Opening PowerShell

  1. Press the Windows key on your keyboard (the one with the Windows logo)
  2. Type PowerShell
  3. Click on "Windows PowerShell" in the search results
  4. A blue window opens with a blinking cursor — that's PowerShell!

You'll see something like PS C:\Users\YourName> followed by a blinking cursor. This is where you type commands.

You opened Terminal!

If you see a window with a blinking cursor waiting for you to type, you're ready for the next step. Leave this window open.

Part 2: Install OpenCode

Mac Installation

  1. Click the "Copy command" button below
  2. Go to your Terminal window
  3. Click inside the Terminal window to make sure it's active
  4. Paste the command: press Cmd + V
  5. Press Enter to run it
curl -fsSL https://opencode.ai/install | bash

You'll see text scrolling as it downloads and installs. Wait until you see your command prompt again (the % or $ symbol).

Windows Installation

  1. Click the "Copy command" button below
  2. Go to your PowerShell window
  3. Click inside the PowerShell window to make sure it's active
  4. Right-click to paste (or press Ctrl + V)
  5. Press Enter to run it
irm get.scoop.sh | iex; scoop install opencode

This installs Scoop (a package manager) first, then OpenCode. Wait until you see your prompt again.

Prefer a Desktop App?

If terminal feels too intimidating, download the desktop app instead: opencode.ai/download

Part 3: Connect to AI Models (Free Option Available)

OpenCode needs to connect to AI models. The easiest way is using OpenCode Zen - free, curated models that work out of the box.

Option A: OpenCode Zen (Easiest - Free)

  1. When OpenCode starts, type /connect and press Enter
  2. Select "opencode" from the list
  3. It will open opencode.ai/auth in your browser
  4. Sign in with Google or GitHub
  5. Done! You're connected to free AI models

Option B: OpenRouter (More Models)

  1. Go to openrouter.ai and sign up
  2. Go to openrouter.ai/keys and create a key
  3. In OpenCode, type /connect and select "openrouter"
  4. Paste your API key when prompted

Recommendation

Start with OpenCode Zen (Option A). It's free and works great. You can always add other providers later.

Part 4: Configure OpenCode

Run OpenCode for the first time:

opencode

OpenCode will ask for your API key. Paste the key you copied from OpenRouter and press Enter.

Part 5: Your First OpenCode Session

Navigate to your Desktop (or any folder you want to work in):

cd ~/Desktop

Start OpenCode:

opencode

Now try asking it to create something:

Create a simple hello world HTML page with a button that changes the background color when clicked

OpenCode will create the file directly on your Desktop. Open it in your browser to see the result!

Congratulations!

You've set up OpenCode and created your first file using terminal-based AI coding!

Essential Terminal Commands

Navigate your computer like a pro

Before using OpenCode effectively, learn these basic terminal commands:

Command What It Does Example
cd Change directory (folder) cd ~/Desktop
ls List files in current folder ls
pwd Show current folder path pwd
mkdir Create a new folder mkdir myproject
clear Clear the terminal screen clear

Common Paths

  • ~ = Your home folder
  • ~/Desktop = Your Desktop
  • ~/Documents = Your Documents folder
  • .. = Go up one folder

Pro Tip: Tab Completion

Press Tab while typing a folder or file name to auto-complete it. This saves typing and prevents typos!

Troubleshooting

Common problems and solutions

Pro Tip: Use AI for Troubleshooting

If you get an error, paste it into ChatGPT or Claude and ask for help. AI is great at explaining terminal errors!

Common Issues

"command not found: opencode"

The installation didn't complete, or you need to restart your terminal. Close Terminal and open it again, then try opencode.

"Permission denied"

On Mac, you may need to allow the app in System Preferences > Security & Privacy. On Windows, try running PowerShell as Administrator.

"Invalid API key"

Make sure you copied the entire key from OpenRouter, including the sk-or- prefix. You can reconfigure with opencode --config.

"curl: command not found" (Mac)

You may need to install Xcode Command Line Tools. Run: xcode-select --install

Terminal is scary / I'm lost

That's normal! Take it slow. Type pwd to see where you are, ls to see what's there, and cd ~/Desktop to go somewhere familiar.
